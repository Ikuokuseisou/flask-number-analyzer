# Flask Number Analyzer

Flaskの学習の一環として作成した簡単なAPIです。
クエリパラメータで受け取った整数列を解析し、統計情報をJSONで返します。

## 作った理由
- Flaskの基本的なルーティングを理解するため
- パラメータ処理とエラーハンドリングの練習
- APIとしてJSONで返す練習

## 機能
- 合計 / 平均 / 最大 / 最小
- ソート
- 中央値
- 最頻値
- threshold以上の件数と合計

## 工夫した点
- 必須パラメータがない場合は400を返すようにした
- 数値変換エラーをtry/exceptで処理した
- 計算ロジックを別ファイルに分けた

## 実行方法
pip install flask
python app.py
